# flo Overflow
Live Site: https://flo-overflow.herokuapp.com/

## Welcome to flo Overflow!

flo Overflow is a website designed to let users ask and answers questions regarding music!

![homepage](https://cdn.discordapp.com/attachments/904438142063501353/910576059819364413/unknown.png)

## Technologies Used
flo Overflow was build using JavaScript, HTML, and CSS. The backend core server and database were created using Sequelize and Express, respectively. The PUG template was used to create the HTML front end and styling was done through the usage of CSS. Additionally, JavaScript scripts were used to make features on the website more dynamic.

## Usage

* After the user signs up or logs in, they will be redirected to their user page that shows all of the questions they've asked

![user_homepage](https://cdn.discordapp.com/attachments/904438142063501353/910574408995848252/unknown.png)

* They can click on the question, which should redirect the user to the specific question page, where they can edit and delete their question

<img src="https://cdn.discordapp.com/attachments/904438142063501353/910575247294607410/unknown.png" alt="user_question_page" width="75%">

* On the user's homepage, the user can navigate to the all questions page, which will redirect the user to the page that contains all of the questions, or can navigate to the Ask A Question page that will redirect the user to a form page to ask a question.
  * User can go to a specific user page and add an answer
  * User's can edit and delete questions and answer's they've created.
